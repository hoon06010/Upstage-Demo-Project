# 🤝 Solar Comm-Coach: 사내 커뮤니케이션 AI 코치

> **"기술은 인간을 대체하는 것이 아니라, 더 나은 동료로 만들어야 합니다."**
> Upstage Solar Pro 3 기반의 비식별화 보안 레이어가 적용된 사내 소통 교정 솔루션입니다.

## 🌟 프로젝트 개요
본 프로젝트는 **Upstage AI Ambassador** 데모를 위해 기획되었습니다. 조직 내 심리적 안전감을 저해하는 공격적인 대화 습관을 진단하고, 심리학적 프레임워크를 바탕으로 건강한 소통 방식을 제안합니다.

## ✨ 핵심 기능 (Key Features)
- **Solar Pro 3 심층 분석:** 한국어의 미묘한 뉘앙스를 파악하여 말투의 공격성을 정확히 진단합니다.
- **PII Masking 보안 레이어:** 외부 API 전송 전 이름, 직급 등 민감 정보를 자동으로 가명화하여 기업 보안 리스크를 해소합니다.
- **전문 코칭 프레임워크:** SBI(상황-행동-영향), NVC(비폭력 대화), 나-대화법(I-Message)을 적용한 교정안을 제공합니다.
- **HCI 기반 인터랙션:** 실시간 스트리밍 출력과 '커뮤니케이션 온도' 시각화를 통해 직관적인 UX를 제공합니다.

## 🛠 기술 스택 (Tech Stack)
- **Language:** Python 3.9+
- **AI Model:** Upstage Solar Pro 3
- **Framework:** Streamlit
- **Security:** Token-based Masking Utility

## 📁 폴더 구조
- `core/`: AI 엔진, 프롬프트 및 포매터 로직
- `utils/`: 데이터 보안(Masking) 및 헬퍼 함수
- `app.py`: 서비스 메인 인터페이스
